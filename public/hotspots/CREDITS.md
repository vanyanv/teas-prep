# Hot-spot diagram credits

All diagrams in this folder are **original schematic illustrations** authored for
this app. They are not derived from any third-party image, so there are no
licensing restrictions (treat as CC0 / public domain).

| File | Subject | Labeled regions |
|------|---------|-----------------|
| `heart.svg` | Heart chambers (anterior schematic) | A right atrium, B left atrium, C right ventricle, D left ventricle |
| `neuron.svg` | Neuron | A dendrites, B cell body (soma), C axon, D myelin sheath, E axon terminals |
| `digestive.svg` | Digestive tract | A esophagus, B stomach, C liver, D small intestine, E large intestine |
| `cell.svg` | Animal cell | A nucleus, B mitochondrion, C cell membrane, D cytoplasm |

Region coordinates (percent of the SVG viewBox) live alongside each HOT_SPOT
question in `src/content/questions.generated.hotspots.ts`.
